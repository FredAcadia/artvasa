# KartPro - Drupal Commerce Theme
KartPro is a clean, elegant, modern and fully responsive Drupal 9, 10 theme. This theme is Drupal commerce ready.

## THEME FEATURES
- Drupal 9.x, 10.x compatible
- Drupal commerce ready.
- Fully responsive
- Clean & modern design
- HTML5 & CSS3
- Self hosted Google Fonts
- Multi-level drop down main menu
- Bootstrap font icons
- Inbuilt slider and banners.
- One column, two column, three column page layout.
- Customizable theme setting
- and more..
